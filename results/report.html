【発展課題】MPI プロセス数（Grid=S）

コンパイル手順:

  1. Makefile.sample を Makefile にコピーし、CC = mpicc を CC = mpicc -cc=icx に変更
  2. CFLAGS = -O3 を CFLAGS = -O3 -std=gnu89 に変更
  3. プロセス数に応じて ./paramset.sh S <x> <y> <z> を実行
  4. make でビルド

実行コマンド（例）:

module load intelmpi/2025
./paramset.sh S 2 2 2
make
mpirun -np 8 ./bmt
